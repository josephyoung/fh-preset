# fh-preset

A preset for vue-cli 4.x

## Features

- project skeleton
- eslint, prettier setting
- dependencies added

## usage

### remote

```bash
$ vue create --preset josephyoung/fh-preset my-project
```

#### local

```bash
$ git clone https://github.com/josephyoung/fh-preset.git
$ vue create --preset ./fh-preset my-project
```
